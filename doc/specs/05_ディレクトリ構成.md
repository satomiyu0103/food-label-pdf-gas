## 主要パスの責務

| パス | 責務 |
|---|---|
| `gas/src/` | **本番アプリ** — GAS ソース。clasp push でデプロイ |
| `gas/src/schema.js` | 商品DB 列定義・Gemini JSON キー（正本） |
| `doc/specs/商品PDF_引継ぎ資料.md` | 引継ぎ・設計たたき台 |

| `doc/reference/migration/meishi-to-product.md` | 名刺PDF版からの移行チェックリスト |

## 命名規則

| 対象 | ルール | 例 |
|---|---|---|
| GAS モジュール | `snake_case.js` または機能名 | `gemini.js`, `main.js` |
| GAS 公開関数 | camelCase | `processAllPendingPdfs` |
| GAS 内部ヘルパー | camelCase + 末尾 `_` | `extractProductFromPdf_` |
| JSON / シート列キー | snake_case（`schema.js` 正本） | `product_name`, `jan_code` |
| Script Properties キー | SCREAMING_SNAKE_CASE | `GEMINI_API_KEY` |
