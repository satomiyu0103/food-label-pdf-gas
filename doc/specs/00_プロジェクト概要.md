# プロジェクト概要

最終更新: 2026年6月28日

---

## このリポジトリについて

**商品 PDF から Google Sheets へデータを取り込む RPA ツール（運用名: SmartShelf）**

Google ドライブ・GAS・Gemini API・スプレッドシートを組み合わせ、商品ラベル・食品表示 PDF から商品名・期限・栄養成分などを自動抽出して「商品DB」シートへ蓄積する。  
アプリ本体は **`gas/src/`**（GAS）に実装する。

---

## 目的（TO-BE）

- 商品 PDF の手入力を排除し、商品データをスプレッドシートへ自動蓄積する
- 要件・設計・実装状況を `doc/specs/` で一元管理する
- エージェントは [`.cursor/rules/agent_core.mdc`](../../.cursor/rules/agent_core.mdc) に従って作業する

## スコープ

| 項目 | 内容 |
|---|---|
| 製品名（GAS / GCP） | SmartShelf |
| リポジトリ名 | food-label-pdf-gas（Git・clone フォルダ・`package.json` の name） |
| 対象ユーザー | 商品情報をスプレッドシートへ取り込む担当者 |
| 技術スタック | GAS / Gemini API / Google スプレッドシート / Slack / clasp |
| 初期前提 | 1 PDF = 1 商品 = 1 シート行 |
| 非スコープ | 在庫管理・販売管理、人手確認ワークフロー、OCR 画像補正 |
