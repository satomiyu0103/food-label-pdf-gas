# Django UI/UX ガイド

> 最終更新: {{LAST_UPDATED}}

Django プロジェクト向けの UI/UX・テンプレート方針を記載する。
技術スタック非依存のテンプレートでは、このファイルは記入式のプレースホルダです。

## 記入項目（例）

- レイアウト基準（ベーステンプレート、ブロック構成）
- コンポーネント・CSS フレームワークの選定
- フォーム・バリデーションエラーの表示ルール
- アクセシビリティ方針

---

Django プロジェクトへ適用する場合は、既存プロジェクトの UI ガイドをこのファイルに転記するか、
`{{PROJECT_CODE}}` 等の参照リポジトリから `Django_UIUX_ガイド.md` をコピーして整備してください。
