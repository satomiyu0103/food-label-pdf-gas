# 既知エラー記録

正本: `doc/ai_guidelines/試験実装のエラー.md`

## 記載する条件（いずれか）

- 再現しうる / 原因が非自明 / 調査 15 分以上 / プロジェクト固有 / 再発防止策を書ける

## 記載しない

タイポ・import 漏れ・個人 PC のみの障害・秘密情報の値

## フォーマット

```
### [YYYY-MM-DD] エラーの概要（1行）

**症状**: ...
**原因**: ...
**対策**: ...
**再発防止**: ...
```

CHANGELOG には `Fixed` + FR コードで要約のみ。
